# EdTech Assignment Tracker - Prototype Submission

## Overview:
A simple assignment tracking system for an EdTech platform allowing:
- ✅ Teachers to create assignments  
- ✅ Students to submit assignments  
- ✅ Teachers to view submissions  

---

## Tech Stack:
- **Backend:** FastAPI, SQLite, JWT Authentication  
- **Frontend:** HTML, CSS, JavaScript (Fetch API)  
- **Database:** SQLite (can be switched to PostgreSQL)  

---

## Features Implemented:
- ✅ User Signup & Login (Role-Based)
- ✅ Assignment Creation (by Teacher)
- ✅ Assignment Submission (by Student)
- ✅ View Submissions (by Teacher)

---

### Backend:
